Earphone Case

Reverse Engineering project

    The purpose of this report is to design an earphone case that is more efficient in many ways. Even though earphone have been used for many years, there are still technical and design flaws in the wide variety of earphone cases. In this project, my teammate and I have redesigned an earphone case with some improvements. The main improvements that have been applied to this design are: minimizing the time that takes users to wrap their earphones properly and maximizing its mobility, so that it can be carried in user’s pocket everywhere.

    Steps taken to bring the design into tangible object

  • 2D design of the project is done using Auto CAD and Intelli CAD

  • 3D design is created by using Creo Parametric

  • The final design is then printed by 3D printer


Professional engineering report has been written explaining failure modes and benefits of the design.
